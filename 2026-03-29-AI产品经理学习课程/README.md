# AI 产品经理学习课程

这套资料面向当前的你：
- 有前端与一定全栈背景
- 做过 `PP@SS`、`COMP9900`、`9321 / 9444 / 9414 / 9517`
- 还没有系统学过产品经理方法论
- 想在较短时间内补齐 `AI 产品经理` 的知识网，并能用于求职与面试

这不是资料堆，而是一套按学习顺序组织的课程包。

## 课程结构

- `00-开始这里`：总导航、学习顺序、你当前的能力差距
- `01-基础课-产品经理底座`：把产品经理的核心概念补齐
- `02-基础课-AI技术扫盲`：把 AI 术语和方法补成能讲清的程度
- `03-提高课-AI产品方法`：学习如何把 AI 能力做成产品
- `04-冲刺课-求职与面试`：把知识转成简历、面试、自我介绍、投递策略与问题应对包
- `05-参考资料`：公开来源与延伸阅读

## 推荐学习顺序

1. 先读 [总导航](./00-开始这里/00-总导航.md)
2. 再读 [你离 AIPM 还差什么](./00-开始这里/02-你离AIPM还差什么.md)
3. 基础课按顺序读完
4. 再进入提高课
5. 最后用冲刺课准备面试与投递
6. 进入 `06-面试问题应对包`，按题型和项目反复训练

## 使用原则

- 学习时始终把你的项目带进去理解，不要把知识和项目割裂
- 每学完一篇，都要问自己三件事：
  - 这个概念在我的哪个项目里出现过？
  - 如果面试官问我，我能不能举例讲清？
  - 它对应的产品产物或交付物是什么？
- 不要追求一次性全懂，先形成知识网，再逐层加深

## 最后提醒

这套课程不能替代真实面试训练。

真正决定你能不能把这些内容用出来的，不是“读过多少篇”，而是：
- 你能不能把概念讲成人话
- 你能不能把项目讲成交付结果
- 你能不能在被追问时继续讲清边界、规则和评估
